Дача Абай — сайт бронирования
==============================

Структура проекта
------------------
index.html      — главная (Hero, преимущества, форма бронирования, шаги, CTA)
about.html      — о даче, что входит в стоимость, калькулятор цены, FAQ
gallery.html    — галерея фото и отзывы гостей
contacts.html   — контакты, карта, погода
style.css       — единая дизайн-система (шрифты, цвета, анимации)
script.js       — вся логика сайта (модалка, калькулятор, FAQ, галерея, бургер-меню и т.д.)
img/            — реальные фотографии дачи (уже подключены)

Фотографии
-----------

  img/46a87e9f-25e1-43c9-b0d0-d596c316aee6.jpg  — бассейн (используется в Hero)

  img/544ee8f8-6926-4258-8ec1-f348be6acf35.jpg  — двор / фасад дома

Чтобы заменить или добавить фото:
  1. Положите новый файл в папку img/.
  2. Обновите путь в нужном месте HTML (background-image в .hero / .gallery-item,
     либо src у <img> в about.html).
  3. Названия файлов и расширения должны совпадать точно (регистр важен).

Рекомендации по фото:
  - Формат JPG или WEBP, ширина от 1600px для чётких фото на Retina-экранах.
  - Сжимайте фото перед загрузкой (например, через squoosh.app), чтобы сайт
    оставался быстрым.
  - Для Hero лучше горизонтальные, крупные снимки.

Контактная форма
-----------------
Кнопка "Отправить заявку" в модальном окне (index.html) открывает Telegram
(t.me/timenub) с готовым сообщением, куда автоматически подставляются имя
и телефон, введённые в форму. Поле телефона принимает только цифры и
разрешённые символы (+, пробел, скобки, дефис).

Публикация (GitHub Pages)
--------------------------
1. Залейте все файлы (index.html, about.html, gallery.html, contacts.html,
   style.css, script.js, img/, robots.txt, sitemap.xml) в корень репозитория.
2. В настройках репозитория включите GitHub Pages → Branch: main → / (root).
3. После публикации замените домен-плейсхолдер "dacha-abay.uz" в файлах
   robots.txt и sitemap.xml на реальный адрес вашего сайта.

SEO
----
robots.txt   — разрешает индексацию всем поисковым роботам, ссылается на sitemap.xml
sitemap.xml  — список всех 4 страниц сайта для поисковых систем
