Как добавить реальные фотографии
=================================

Сайт уже вёрстан под фотографии — сейчас на их месте показывается
красивая градиентная заглушка с подписью, чтобы страница выглядела
готовой ещё до загрузки настоящих снимков.

Чтобы подключить свои фото, просто положите файлы в эту папку
(assets/images/) с такими именами — они подхватятся автоматически:

  hero.jpg           — главное фото на первом экране (горизонтальное, крупное)
  og-cover.jpg        — обложка для соцсетей (1200x630)
  about-1.jpg         — фото двора/территории (блок «О нас»)
  about-2.jpg         — фото бассейна (блок «О нас»)
  card-main.jpg       — фото для карточки дачи
  gallery-1.jpg … gallery-6.jpg — 6 фото для галереи
  video-poster.jpg    — обложка для блока «Видео»

Рекомендации:
- Формат JPG или WEBP, ширина от 1600px для чётких фото на Retina-экранах.
- Сжимайте фото перед загрузкой (например, через squoosh.app), чтобы
  сайт оставался быстрым.
- Названия файлов и расширения должны совпадать точно (регистр важен).
